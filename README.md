# Enterprise Conversational Architectures for E-commerce

This repository showcases high-level conversational architecture, deterministic state-machine logic, and backend API integration mapping for mid-market digital commerce. 

These designs utilize **Google Cloud Platform (GCP)** infrastructure, specifically **Dialogueflow CX** and **Vertex AI Agent Builder**, to transition unpredictable LLM layers into secure, serverless enterprise capital assets.

## 🏗️ Architectural Frameworks Included

### 1. Multi-Tier Order Modification Agent (State-Machine Design)
An enterprise architecture managing complex post-purchase user intents (e.g., partial cancellations, address modifications mid-transit) via custom webhook hooks.
* **Core Tech:** Dialogueflow CX, Cloud Functions, Shopify Admin API.
* **Logic Framework:** Deterministic state tracking with integrated fallback logic to prevent conversation loops.
* **Documentation:** See `/blueprints/order-modification/`

### 2. High-Intent Cart Abandonment & Checkout Preservation Agent
An event-driven conversational framework triggered by storefront telemetry to re-engage abandoning high-value users via sub-12ms event execution.
* **Core Tech:** Vertex AI Agent Builder, Dialogflow CX, Klaviyo / CRM Webhooks.
* **Documentation:** See `/blueprints/cart-preservation/`
